날씨 동물 이용약관

최종 수정일: 2026년 4월 7일

1. 서비스 개요

날씨 동물은 실시간 날씨 데이터를 기반으로 동물 캐릭터를 매칭하여 보여주는 엔터테인먼트 서비스입니다.

2. 위치 정보

본 서비스는 정확한 날씨 정보 제공을 위해 사용자의 위치 정보를 활용합니다. 위치 정보는 기상청 API 호출에만 사용되며, 서버에 저장되지 않습니다. 위치 권한을 거부할 경우 서울 기준 날씨가 표시됩니다.

3. 데이터 저장

도감 진행도와 마지막 표시 캐릭터 정보는 사용자 기기의 localStorage에만 저장됩니다. 별도의 서버 수집이나 제3자 전송은 없습니다.

4. 날씨 데이터

날씨 데이터는 기상청 단기예보 API를 통해 제공됩니다. 데이터의 정확성은 기상청에 의존하며, 서비스 장애 시 캐시된 데이터를 표시할 수 있습니다.

5. 캐릭터 및 콘텐츠

캐릭터 일러스트와 대사는 퍼블릭 도메인 작품을 기반으로 하거나 독립 창작된 오마주입니다. 상업적 재배포는 금지됩니다.

6. 면책사항

날씨 동물의 캐릭터 코멘트는 엔터테인먼트 목적이며, 기상 예보의 대체가 아닙니다. 중요한 기상 판단은 기상청 공식 발표를 참고하세요.
